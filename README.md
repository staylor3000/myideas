# 💡 myideas

A personal idea management app for capturing, organising, and visualising your ideas — built as a Python learning project.

## What it does

- **Mindmap view** — visualise your ideas as connected nodes on an interactive canvas
- **Grid view** — browse ideas as cards with colour-coded categories
- **Categories** — tag ideas as product, design, tech, or business
- **Status tracking** — move ideas from raw → fleshed out → active
- **Quick capture** — add new ideas via a simple modal form
- **Search & filter** — find ideas instantly by keyword or category

## Current state

The current prototype (`ideamap.html`) is a self-contained frontend built in vanilla HTML, CSS, and JavaScript. It runs in the browser with no backend — data is in-memory only.
